# SOC Threat Hunting Labs

This repository contains practical KQL queries used to detect real-world attack techniques across endpoints, identity, and cloud environments using Microsoft Defender XDR.

## Focus Areas
- PowerShell abuse detection
- Identity-based attacks (Impossible Travel, brute force)
- Suspicious process execution chains

## Tools
- Microsoft Defender XDR
- Kusto Query Language (KQL)
- Azure / Entra ID Logs

## Goal
To build detection logic that mirrors real attacker behavior and improves security visibility in modern environments.
